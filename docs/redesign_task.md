重设计任务：有机科幻 / 软 UI
目标
重新设计网站的浅色模式，使其符合“有机科幻”和“软 UI”美学，灵感来源于 PlayStation 5、电影《她》和《机器人总动员》中的伊娃 (EVE)。

视觉语言
形状
无锐角：使用大边框圆角（例如 rounded-3xl、rounded-[2rem]）。
形态：鹅卵石状、胶囊状、泪滴状和流线型。
材质与纹理
陶瓷白：纯白或米白色表面，带微妙的内发光（shadow-inner-white）。
磨砂玻璃：使用 backdrop-blur 实现分层效果。
无边框：用柔和的阴影（shadow-lg、shadow-soft）和微妙的光照效果替代深色边框。
动效
背景：缓慢浮动的模糊彩色光球（呼吸效果）。
加载：平滑脉冲或旋转的环形动画。
排版
字体：圆润的几何无衬线字体 (Outfit)。
颜色：文本使用深灰色（text-zinc-800），而非纯黑色。
实施计划
字体：

更新 lib/font.ts，使用 next/font/google 中的 Outfit 字体。
全局样式与主题：

更新 tailwind.config.cjs 以扩展主题：
将 Outfit 添加到字体家族。
定义“陶瓷”阴影工具类。
定义浮动光球动画的关键帧。
更新 app/globals.css 以包含浮动光球动画。
布局更新：

修改 app/(main)/layout.tsx：
移除网格背景。
添加浮动光球背景容器。
更新主容器为“陶瓷白”（高透明度白色，带模糊效果，如果适用则使用圆角，尽管主容器可能占据全高，具体视情况而定）。用户提到了“有机科幻”，这通常意味着内容位于卡片上，或者页面本身就是画布。当前的布局使用一个中央列 bg-zinc-50/90。我将把这个列做得更像“玻璃”或“陶瓷”质感，并移除 ring。
组件样式：

按钮：更新为药丸形状，柔和阴影，白/灰渐变或纯白色，并带有悬停抬升效果。
卡片：更新为大边框圆角，白色背景，柔和阴影，无边框。
导航：更新为浮动胶囊状或玻璃质感条。
页脚：简洁，圆润。
验证：

检查浅色模式的外观。
确保深色模式未被破坏（或仅进行最小限度适配）。
